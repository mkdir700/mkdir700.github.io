167. 两数之和 II - 输入有序数组

Posted on 2020-08-08 | In 算法题

给定一个已按照升序排列 的有序数组，找到两个数使得它们相加之和等于目标数。

函数应该返回这两个下标值 index1 和 index2，其中 index1 必须小于 index2。

说明:

  • 返回的下标值（index1 和 index2）不是从零开始的。
  • 你可以假设每个输入只对应唯一的答案，而且你不可以重复使用相同的元素。

示例:

输入: numbers = [2, 7, 11, 15], target = 9

输出: [1,2]

解释: 2 与 7 之和等于目标数 9 。因此 index1 = 1, index2 = 2 。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/two-sum-ii-input-array-is-sorted

二分查找

思路

target是数组中两个必定存在的数相加而来

我们可以固定一个数x，然后通过「二分查找」找出另外一个数字y

如果没有找到，则重新固定一个数，继续寻找

举例：

[2, 7, 11, 15] target = 9

target - numbers[0] 等于 7，然后在区间[1 ... len(numbers)-1]内 使用二分查找法，寻找是否有等于7的元素

代码

class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        size = len(numbers)
        for i in range(size):
            # 待查找的值
            y = target - numbers[i]
            # 确定范围
            left, right = i + 1, size - 1
            # 二分查找
            while left <= right:
                mid = (left + right) >> 1
                if numbers[mid] < y:
                    left = mid + 1
                elif numbers[mid] > y:
                    right = mid - 1
                else:
                    return [i+1, mid+1]

双指针

思路

为数组的第一个元素和最后一个元素分别定义指针 l h

  • numbers[l] + numbers[h] > target

相加值大于target，此时我们需要让这个值变小，所以应该使h指针向左移动

  • numbers[l] + numbers[h] < target

相加值小于target，此时我们需要让这个值变大，所以应该使l指针向右移动

  • numbers[l] + numbers[h] == target

找到答案，返回[l+1, h+1]即可

代码

class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        l, h = 0, len(numbers) - 1
        while l < h:
            if numbers[l] + numbers[h] > target:
                h -= 1
            elif numbers[l] + numbers[h] < target:
                l += 1
            else:
                return [l+1, h+1]
